A lightweight utility tool that toggles the terrain grid without all the extra clutter from the normal Layers tool.

### ✨ Features

- 🗺️ Shows only the **Terrain Grid** overlay.
- 🔒 Does **not** get deactivated when pressing `Escape`.
- ⚡ Provides the same result as opening the Layers tool and disabling every option except **Terrain Grid**.
- ⌨️ Shortcut: `Shift + F`
